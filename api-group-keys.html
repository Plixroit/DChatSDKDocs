Group Key Operations

Symmetric group keys and sender key management for group messaging.

Overview

Group chats use two types of symmetric keys:

  • Group Key: Shared symmetric key for the entire group (legacy, being deprecated)
  • Sender Key: Per-member-per-group key for sender-key encryption (recommended)

Sender keys allow each member to encrypt messages with their own key, avoiding the need to distribute a new group key on every member change.

Group Keys (Legacy)

StoreGroupKeyAsync Task

Stores a symmetric group key keyed by group ID.

ParameterTypeDescription
groupIdstringGroup identifier
symmetricKeyBase64stringAES-256 key (base64)
keyGenerationintKey version number
await sdk.StoreGroupKeyAsync(groupId, symmetricKeyBase64, keyGeneration: 1);
GetGroupKeyAsync Task<string?>

Retrieves a symmetric group key by group ID.

string? key = await sdk.GetGroupKeyAsync(groupId);
DeleteGroupKeyAsync Task

Deletes a stored symmetric group key.

await sdk.DeleteGroupKeyAsync(groupId);

Sender Keys (Recommended)

GenerateSenderKeyAsync Task<string>

Generates a new sender key for a group member. Returns base64-encoded AES-256 key.

ParameterTypeDescription
ownerDidstringDID of the key owner
groupIdstringGroup identifier
string senderKey = await sdk.GenerateSenderKeyAsync(ownerDid, groupId);
StoreSenderKeyAsync Task

Stores a sender key for a specific member/group tuple.

ParameterTypeDescription
ownerDidstringDID of the key owner
groupIdstringGroup identifier
senderKeyBase64stringAES-256 key (base64)
await sdk.StoreSenderKeyAsync(ownerDid, groupId, senderKeyBase64);
GetSenderKeyAsync Task<string?>

Retrieves a sender key for a specific member/group tuple.

string? key = await sdk.GetSenderKeyAsync(ownerDid, groupId);
DeleteSenderKeyAsync Task

Deletes a sender key for a specific member/group tuple.

await sdk.DeleteSenderKeyAsync(ownerDid, groupId);
IncrementSenderKeyMessageCountAsync Task<bool>

Increments message count and checks if sender key rotation is needed. Returns true if rotation threshold reached.

bool needsRotation = await sdk.IncrementSenderKeyMessageCountAsync(ownerDid, groupId);
if (needsRotation) {
    // Rotate and redistribute
}
RotateSenderKeyAsync Task<bool>

Rotates a sender key and distributes the new key to all group members.

ParameterTypeDescription
groupIdstringGroup identifier
memberDidstringDID of member whose key is rotating
reasonstringReason for rotation
await sdk.RotateSenderKeyAsync(groupId, memberDid, "scheduled");
