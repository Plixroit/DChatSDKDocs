DChat SDK Configuration

Configuration

All SDK behaviour is controlled via DChatSdkOptions, built with a fluent builder. This page documents every option and how to choose its value.

DChatSdkOptionsBuilder

Create an options instance with the static factory DChatSdkOptions.CreateBuilder(). The builder validates required fields and throws InvalidOperationException if mandatory options are missing when you call Build().

DChatSdkOptions options = DChatSdkOptions.CreateBuilder()
    .AddNode(new NodeConfiguration(
        blockchainRpcUrl: "https://rpc.dchat.network",
        grpcServerUrl:    "https://grpc.dchat.network",
        middlewareApiUrl: "https://api.dchat.network",
        name:             "mainnet"))
    .SetChatKeyRotationPolicy(new KeyRotationPolicy
    {
        MaxMessageCount = 5000,
        MaxAgeDays      = 14
    })
    .SetGrpcMaxMessageSize(8 * 1024 * 1024)  // 8 MB
    .SetGroupMemberProvider(myProvider)
    .Build();
When AddNode() is used, the builder auto-fills blockchainRpcUrl, grpcServerUrl, and middlewareApiUrl from the first node if they have not been set explicitly. You can override any individual URL after calling AddNode().

Builder Methods

DChatSdkOptionsBuilder SetMiddlewareApiUrl(string middlewareApiUrl)

Sets the base URL for the authenticated middleware API. Used for IPFS uploads, identity lookups, and profile operations. Overrides the URL sourced from AddNode().

DChatSdkOptionsBuilder SetGrpcServerUrl(string grpcServerUrl)

Sets the gRPC server URL for the bidirectional message stream. Must be reachable from the device and support HTTP/2.

DChatSdkOptionsBuilder SetBlockchainRpcUrl(string rpcUrl)

Sets the Cosmos/CometBFT RPC endpoint for identity registration and on-chain queries. Required for username registration and DID lookup.

DChatSdkOptionsBuilder SetGrpcMaxMessageSize(int maxSize)

Maximum gRPC message size in bytes. Defaults to 4 MB. Increase to 816 MB if you plan to send large file chunks directly over the stream.

ParameterTypeDescription
maxSizeintByte limit (must be > 0). Default: 4194304.
DChatSdkOptionsBuilder SetGrpcKeepAliveInterval(TimeSpan interval)

How often the gRPC channel sends HTTP/2 PING frames to keep NAT mappings alive. Default: 60 s. Reduce to 2030 s on mobile networks with aggressive NAT timeouts.

DChatSdkOptionsBuilder SetGrpcConnectionTimeout(TimeSpan timeout)

Maximum time to wait when establishing the gRPC connection. Default: 30 s.

DChatSdkOptionsBuilder SetChatKeyRotationPolicy(KeyRotationPolicy policy)

Configures when E2EE chat keys are automatically rotated for forward secrecy. See Encryption Key Rotation for full details.

DChatSdkOptionsBuilder SetGroupMemberProvider(IGroupMemberProvider provider)

Injects your application's group-membership resolver. Required for group messaging. The SDK calls this when distributing sender keys after a rotation. See Groups.

DChatSdkOptionsBuilder AddNode(NodeConfiguration node)

Registers a DChat network node. The builder uses the first added node to auto-populate blockchainRpcUrl, grpcServerUrl, and middlewareApiUrl if they are not set individually.

DChatSdkOptionsBuilder SetDefaultDerivationPath(string derivationPath)

BIP44 derivation path used when deriving keys from a mnemonic. Default: m/44'/118'/0'/0/0 (Cosmos coin type). Change only if targeting a different slip-0044 coin type.

DChatSdkOptionsBuilder SetDefaultPrefix(string prefix)

Bech32 human-readable part (HRP) used when encoding wallet addresses. Default: "dc", producing addresses like dc1q....

NodeConfiguration

Groups the three endpoints that describe a single DChat network node.

PropertyTypeDescription
NamestringDisplay name (defaults to the hostname of BlockchainRpcUrl).
BlockchainRpcUrlstringCosmos/CometBFT RPC endpoint for on-chain operations.
GrpcServerUrlstringgRPC messaging endpoint (HTTP/2).
MiddlewareApiUrlstringREST middleware for IPFS uploads, authenticated queries.
var node = new NodeConfiguration(
    blockchainRpcUrl: "https://rpc.dchat.network",
    grpcServerUrl:    "https://grpc.dchat.network",
    middlewareApiUrl: "https://api.dchat.network",
    name:             "mainnet");

KeyRotationPolicy

Controls the automatic rotation threshold for E2EE chat keys. Rotation is triggered when either threshold is exceeded.

PropertyTypeDefaultDescription
MaxMessageCountlong10 000Rotate the key after this many encrypted messages.
MaxAgeDaysint30Rotate the key after this many days regardless of message volume.
Lower thresholds provide better forward secrecy at the cost of more key-rotation handshake messages. For high-volume bots or channels, consider a higher MaxMessageCount.
