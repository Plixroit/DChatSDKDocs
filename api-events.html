DChat SDK API Reference Events

SDK Events

All events raised by IDChatSDKClient for real-time notifications.

All events are raised on the thread pool. Marshal to the UI thread as required by your framework (e.g., Dispatcher.Dispatch in MAUI, InvokeOnMainThread in Xamarin).

Connection Events

OnConnected EventHandler Fires when the stream is successfully established or re-established after a disconnect.
OnDisconnected EventHandler Fires when the stream closes (graceful or error).

Message Lifecycle Events

OnMessageSent EventHandler<QueuedMessage> Fires when a message has been written to the stream (before ACK).
OnMessageFailed EventHandler<QueuedMessage> Fires when a message times out waiting for an ACK. The message is moved to the failed cache for retry.
OnMessageStatusChanged EventHandler<MessageStatusUpdatedEventArgs> Fires when a message status changes (sent, delivered, read).
OnMessageEditReceived EventHandler<MessageEditReceivedEventArgs> Fires when a message edit is received.
OnMessageDeleteReceived EventHandler<MessageDeleteReceivedEventArgs> Fires when a message delete is received.
OnMessagePinReceived EventHandler<MessagePinReceivedEventArgs> Fires when a message pin/unpin is received.

Typed Message Events

OnMessageReceived EventHandler<DecryptedMessage> Fires when a non-typed message is received (presence, system, fallback messages).
OnTextMessageReceived EventHandler<TextMessageReceivedEventArgs> Fires when a text message is received.
OnImageMessageReceived EventHandler<ImageMessageReceivedEventArgs> Fires when an image message is received.
OnVideoMessageReceived EventHandler<VideoMessageReceivedEventArgs> Fires when a video message is received.
OnAudioMessageReceived EventHandler<AudioMessageReceivedEventArgs> Fires when an audio message is received.
OnLocationMessageReceived EventHandler<LocationMessageReceivedEventArgs> Fires when a location message is received.

Contact & Profile Events

OnContactRequestReceived EventHandler<ContactRequestReceivedEventArgs> Fires when a contact request is received.
OnContactResponseReceived EventHandler<ContactResponseReceivedEventArgs> Fires when a contact response is received (accept/decline/block).
OnProfileRequestReceived EventHandler<ProfileRequestReceivedEventArgs> Fires when a profile request is received.
OnProfileUpdateReceived EventHandler<ProfileUpdateReceivedEventArgs> Fires when a profile update is received.
OnContactPresenceRequestReceived EventHandler<ContactPresenceRequestReceivedEventArgs> Fires when a presence request is received.
OnContactPresenceResponseReceived EventHandler<ContactPresenceResponseReceivedEventArgs> Fires when a presence response is received.

Key Management Events

OnKeyRotationReceived EventHandler<KeyRotationMessageData> Fires when a key rotation message is received from a contact.

Group Events

OnGroupCreateReceived EventHandler<GroupCreateReceivedEventArgs> Fires when a group creation message is received.
OnGroupMemberAddReceived EventHandler<GroupMemberAddReceivedEventArgs> Fires when a group member add is received.
OnGroupMemberRemoveReceived EventHandler<GroupMemberRemoveReceivedEventArgs> Fires when a group member remove is received.
OnGroupAdminUpdateReceived EventHandler<GroupAdminUpdateReceivedEventArgs> Fires when a group admin update is received (promote/demote).
OnGroupKeyRotationReceived EventHandler<GroupKeyRotationReceivedEventArgs> Fires when a group key rotation is received.
OnGroupNameChangeReceived EventHandler<GroupNameChangeReceivedEventArgs> Fires when a group name change is received.
OnGroupDescriptionChangeReceived EventHandler<GroupDescriptionChangeReceivedEventArgs> Fires when a group description change is received.
OnGroupPhotoChangeReceived EventHandler<GroupPhotoChangeReceivedEventArgs> Fires when a group photo change is received.
OnGroupPhotoRemoveReceived EventHandler<GroupPhotoRemoveReceivedEventArgs> Fires when a group photo remove is received.
OnGroupMetadataRequestReceived EventHandler<GroupMetadataRequestReceivedEventArgs> Fires when a group metadata request is received.
OnGroupStateHashRequestReceived EventHandler<GroupStateHashRequestReceivedEventArgs> Fires when a group state hash request is received.
OnGroupStateHashResponseReceived EventHandler<GroupStateHashResponseReceivedEventArgs> Fires when a group state hash response is received.

Subscribing to Events

// Subscribe to events after creating the SDK client
sdk.MessageReceived += OnMessageReceived;
sdk.TextMessageReceived += OnTextMessage;
sdk.ImageMessageReceived += OnImageMessage;
sdk.ContactRequestReceived += OnContactRequest;
sdk.OnConnected += OnConnected;
sdk.OnDisconnected += OnDisconnected;

// Handler example
void OnTextMessage(object? sender, TextMessageReceivedEventArgs e)
{
    // Marshal to UI thread if needed
    MainThread.BeginInvokeOnMainThread(() => {
        var msg = e.Message;
        DisplayMessage(msg.SenderDid, msg.Content);
    });
}

// Unsubscribe when disposing
sdk.MessageReceived -= OnMessageReceived;
