Network & Node Operations

Manage node connections, blockchain validation, and faucet funding.

Node Management

CurrentNode NodeConfiguration?

Gets the current active node configuration.

NodeConfiguration? node = sdk.CurrentNode;
Console.WriteLine(node?.GrpcEndpoint);
AvailableNodes IReadOnlyList<NodeConfiguration>

Gets the list of available node configurations provided during initialization.

foreach (var node in sdk.AvailableNodes) {
    Console.WriteLine($"{node.Name}: {node.GrpcEndpoint}");
}
SwitchNodeAsync Task

Switches the active node configuration. May trigger a reconnection.

ParameterTypeDescription
nodeNodeConfigurationNode configuration to switch to
var newNode = sdk.AvailableNodes.First(n => n.Name == "backup");
await sdk.SwitchNodeAsync(newNode);

Blockchain Operations

ValidateAccountExistsAsync Task<bool>

Validates that an account exists on the blockchain.

ParameterTypeDescription
walletAddressstringWallet address to validate
bool exists = await sdk.ValidateAccountExistsAsync(walletAddress);
if (!exists) {
    // Account not found on blockchain
}
RegisterIdentityAsync Task<BlockchainIdentityResult>

Registers an identity on the blockchain.

ParameterTypeDescription
usernamestringDesired username
keyPairDChatChainKeyPairKey pair for registration
BlockchainIdentityResult result = await sdk.RegisterIdentityAsync(username, keyPair);

Faucet (Testnet)

RequestFundingAsync Task<FaucetResponse>

Requests funding for a key pair from the faucet service (testnet only).

ParameterTypeDescription
keyPairDChatChainKeyPairKey pair to fund
usernamestring?Optional username for memo
FaucetResponse response = await sdk.RequestFundingAsync(keyPair, username);
if (response.Success) {
    Console.WriteLine($"Funded! TX: {response.TransactionHash}");
}

Failed Message Cache

StoreFailedMessageId void

Stores a failed message ID for retry after key exchange.

sdk.StoreFailedMessageId(senderDid, messageId);
GetFailedMessageIds List<string>

Gets all failed message IDs for a sender.

List<string> ids = sdk.GetFailedMessageIds(senderDid);
ClearFailedMessageIds void

Clears specific failed message IDs after successful resend.

sdk.ClearFailedMessageIds(senderDid, messageIds);
ClearAllFailedMessagesForSender void

Clears all failed messages for a sender.

sdk.ClearAllFailedMessagesForSender(senderDid);
ClearUserSession void

Clears all cached keys and session information from memory. Call on user logout.

sdk.ClearUserSession();
