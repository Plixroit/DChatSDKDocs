API Reference

Complete reference for IDChatSDKClient interface, organized by category.

Overview

The IDChatSDKClient interface provides a unified API for all SDK functionality including identity management, messaging, encryption, and blockchain operations. Select a category below to view detailed method signatures, parameters, and usage examples.

API Categories

Connection
Connect, disconnect, connection state properties.
Identity
Create, restore, store, and manage blockchain identities.
Chat Keys
Generate, rotate, and distribute chat encryption keys.
Encryption
ECIES and symmetric encryption operations.
Group Keys
Symmetric group keys and sender key management.
Messaging
Send text, images, videos, files, locations, albums.
Group Messaging
Send messages to groups with sender-key encryption.
Group Control
Create groups, add/remove members, update settings.
Contacts & Profile
Contact requests, profile requests and updates.
Presence
Online status requests and responses.
Media & IPFS
Upload, download, encrypt, and decrypt media.
Network
Node switching, blockchain validation, faucet.
Events
All SDK events for message, connection, and group handling.
