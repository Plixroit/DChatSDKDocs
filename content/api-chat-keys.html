Chat Key Operations

Generate, rotate, and manage chat encryption keys for E2EE messaging.

Overview

Chat keys are X25519 key pairs used for ECIES encryption in 1:1 messaging. Each user has one active chat key at a time, identified by generation number. Keys should be rotated periodically or when compromised.

Key Management

GetActiveChatKeyPairAsync Task<ChatKeyPair?>

Gets the currently active chat key pair for the specified owner.

ParameterTypeDescription
ownerDidstringDID of the key owner
passwordstring?Optional password for decryption
ChatKeyPair? chatKey = await sdk.GetActiveChatKeyPairAsync(ownerDid, password);
if (chatKey != null) {
    Console.WriteLine($"Active key generation: {chatKey.Generation}");
}
GenerateChatKeyPairAsync Task<ChatKeyPair>

Generates a new chat key pair for the specified owner.

ParameterTypeDescription
ownerDidstringDID of the key owner
generationint?Optional generation number (auto-incremented if null)
ChatKeyPair newKey = await sdk.GenerateChatKeyPairAsync(ownerDid, generation: null);
Console.WriteLine($"New public key: {newKey.PublicKeyBase64}");
StoreChatKeyPairAsync Task

Stores a chat key pair securely.

await sdk.StoreChatKeyPairAsync(chatKeyPair, password);

Key Rotation

RotateKeyAsync Task<ChatKeyPair>

Rotates the chat key for the specified owner. Returns the new key pair.

ParameterTypeDescription
ownerDidstringDID of the key owner
reasonstringReason for rotation (e.g., "scheduled", "compromised")
ChatKeyPair newKey = await sdk.RotateKeyAsync(ownerDid, reason: "scheduled");
// Distribute new key to contacts
ShouldRotateKeyAsync Task<bool>

Checks if the active chat key needs rotation based on policy (age, message count, etc.).

if (await sdk.ShouldRotateKeyAsync(ownerDid)) {
    await sdk.RotateKeyAsync(ownerDid, "policy");
}
SendKeyRotationMessageAsync Task<bool>

Sends a key rotation notification to a recipient with the new chat public key.

ParameterTypeDescription
recipientDidstringDID of the recipient
recipientWalletstringWallet address for routing
recipientPubKeystringRecipient's identity public key
newChatPubKeystringNew chat public key to share
await sdk.SendKeyRotationMessageAsync(
    recipientDid, recipientWallet, recipientPubKey, newChatPubKey);
