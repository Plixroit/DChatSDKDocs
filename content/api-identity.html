DChat SDK > API Reference > Identity

Identity Operations

Create, restore, store, and manage blockchain identities and key pairs.

Creating Identities

CreateIdentityAsync Task<BlockchainIdentityResult>

Creates a new identity by generating a key pair, storing it securely, and registering on the blockchain.

ParameterTypeDescription
usernamestringDesired username (3-30 characters)
passwordstring?Optional password for additional encryption
BlockchainIdentityResult result = await sdk.CreateIdentityAsync("alice");
Console.WriteLine(result.Identity.WalletAddress);
Console.WriteLine(result.TransactionHash);
CreateAccountAsync Task<BlockchainIdentityResult>

Creates a new account by requesting funding and registering identity on blockchain. Combines RequestFundingAsync and CreateIdentityAsync.

DChatChainKeyPair keyPair = await sdk.RestoreFromSeedPhraseAsync(seedPhrase);
BlockchainIdentityResult result = await sdk.CreateAccountAsync(keyPair, "bob");
RegisterIdentityAsync Task<BlockchainIdentityResult>

Registers an identity on the blockchain without funding.

BlockchainIdentityResult result = await sdk.RegisterIdentityAsync(username, keyPair);

Looking Up Identities

GetIdentityAsync Task<BlockchainIdentity?>

Gets an identity by wallet address or username.

BlockchainIdentity? identity = await sdk.GetIdentityAsync("alice");
if (identity != null) {
    Console.WriteLine(identity.WalletAddress);
}
CheckUsernameAvailabilityAsync Task<bool>

Checks if a username is available for registration.

bool available = await sdk.CheckUsernameAvailabilityAsync("bob");
if (available) {
    // Username is free to register
}
GetUserProfileUsernameAsync Task<string?>

Gets the blockchain username bound to a wallet address. Returns null if no profile found.

string? username = await sdk.GetUserProfileUsernameAsync(walletAddress);

Seed Phrases

GenerateSeedPhraseAsync Task<string>

Generates a new BIP39 seed phrase (12 or 24 words).

string seed24 = await sdk.GenerateSeedPhraseAsync(wordCount: 24);
string seed12 = await sdk.GenerateSeedPhraseAsync(wordCount: 12);
RestoreFromSeedPhraseAsync Task<DChatChainKeyPair>

Restores a key pair from a BIP39 seed phrase.

ParameterTypeDescription
seedPhrasestringBIP39 mnemonic phrase
derivationPathstringBIP44 path (default: m/44'/118'/0'/0/0)
DChatChainKeyPair keyPair = await sdk.RestoreFromSeedPhraseAsync(
    seedPhrase,
    derivationPath: "m/44'/118'/0'/0/0"
);

Key Pair Storage

StoreKeyPairAsync Task

Stores an identity key pair securely on device.

await sdk.StoreKeyPairAsync(keyPair, password: null);
GetKeyPairAsync Task<DChatChainKeyPair?>

Gets a stored identity key pair by wallet address.

DChatChainKeyPair? kp = await sdk.GetKeyPairAsync(walletAddress, password);
HasKeyPairAsync Task<bool>

Checks if a key pair exists for the given wallet address.

bool hasKey = await sdk.HasKeyPairAsync(walletAddress);
DeleteKeyPairAsync Task

Deletes a stored key pair from device.

await sdk.DeleteKeyPairAsync(walletAddress);
ActivateIdentityAsync Task<bool>

Activates an identity by loading its key pair into the SDK context.

bool activated = await sdk.ActivateIdentityAsync(walletAddress, password);

Signing & Verification

SignMessageAsync Task<string>

Signs a message using the identity key. Returns hex-encoded signature.

string signature = await sdk.SignMessageAsync(message, walletAddress, password);
VerifySignatureAsync Task<bool>

Verifies a signature against a message and public key.

bool valid = await sdk.VerifySignatureAsync(message, signature, publicKey);
