Connection Management

Methods and properties for managing the connection to the messaging service.

ConnectAsync Task<bool>

Connects to the messaging service using the active identity. Returns true if connection succeeded.

bool connected = await sdk.ConnectAsync();
if (connected) {
    Console.WriteLine("Connected successfully");
}
DisconnectAsync Task

Disconnects from the messaging service gracefully. Flushes pending messages before closing.

await sdk.DisconnectAsync();
IsConnected bool

Gets whether the client is currently connected to the messaging service.

if (sdk.IsConnected) {
    // Ready to send messages
}
IsAuthenticated bool

Gets whether the client has an active identity loaded (key pair available).

if (sdk.IsAuthenticated) {
    // Identity loaded, can sign and encrypt
}
