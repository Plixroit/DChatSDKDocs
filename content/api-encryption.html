Encryption Operations

Low-level ECIES and symmetric encryption operations for advanced use cases.

Overview

These methods provide direct access to encryption primitives. In most cases, you should use the high-level messaging methods (e.g., SendTextMessageAsync) which handle encryption automatically. Use these when implementing custom protocols or debugging.

ECIES Encryption (1:1)

EncryptMessageAsync Task<string>

Encrypts a message using ECIES with the recipient's chat public key. Returns JSON envelope.

ParameterTypeDescription
plaintextstringMessage to encrypt
recipientPubKeystringRecipient's X25519 public key (base64)
senderDidstringSender's DID for envelope metadata
string encrypted = await sdk.EncryptMessageAsync(
    plaintext, 
    recipientPubKey, 
    senderDid
);
DecryptMessageAsync Task<string>

Decrypts an ECIES-encrypted message using your chat private key.

ParameterTypeDescription
encryptedJsonstringECIES envelope from EncryptMessageAsync
senderPubKeystringSender's X25519 public key (for verification)
recipientDidstringYour DID (to locate private key)
string plaintext = await sdk.DecryptMessageAsync(
    encryptedJson, 
    senderPubKey, 
    recipientDid
);

Symmetric Encryption (Groups)

EncryptSymmetricAsync Task<string>

Encrypts plaintext using a symmetric group/sender key (AES-256-GCM). Returns base64 ciphertext with prepended nonce.

ParameterTypeDescription
plaintextstringMessage to encrypt
symmetricKeyBase64stringAES-256 key (base64)
string encrypted = await sdk.EncryptSymmetricAsync(
    plaintext, 
    symmetricKeyBase64
);
DecryptSymmetricAsync Task<string>

Decrypts a symmetric envelope using a group/sender key.

ParameterTypeDescription
encryptedPayloadstringBase64 ciphertext from EncryptSymmetricAsync
symmetricKeyBase64stringAES-256 key (base64)
string plaintext = await sdk.DecryptSymmetricAsync(
    encryptedPayload, 
    symmetricKeyBase64
);
Security Note: Never log or transmit encryption keys in plaintext. Use secure storage for private keys and rotate keys periodically.
