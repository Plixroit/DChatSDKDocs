DChat SDK > API Reference > Messaging

Messaging Operations

Send text, images, videos, audio, files, locations, and albums to individual contacts.

Core Send Methods

SendMessageToWalletAsync Task<bool>

Sends a typed message payload to a recipient. Use this for new code - wallet address is used for gRPC routing.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
payloadMessagePayloadTyped message content
recipientPubKeystringRecipient's chat public key
var payload = new MessagePayload { ... };
bool sent = await sdk.SendMessageToWalletAsync(recipientWallet, payload, recipientPubKey);
SendEncryptedEnvelopeToWalletAsync Task<bool>

Sends a pre-encrypted envelope directly. Caller is responsible for encryption.

bool sent = await sdk.SendEncryptedEnvelopeToWalletAsync(recipientWallet, encryptedJson);

Text Messages

SendTextMessageAsync Task<bool>

Sends a text message with optional reply support.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
contentstringMessage text
senderDidstringSender's DID
recipientDidstringRecipient's DID
chatIdstringChat identifier
recipientPubKeystringRecipient's chat public key
replyToMessageIdstring?Optional reply reference
await sdk.SendTextMessageAsync(
    recipientWallet, 
    content, 
    senderDid, 
    recipientDid, 
    chatId, 
    recipientPubKey,
    replyToMessageId: null
);

Album Messages

SendAlbumMessageAsync Task<bool>

Sends a multi-image album as ONE message. Uploads all items to IPFS in parallel.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
itemsList<AlbumImageInput>List of album items
senderDidstringSender's DID
recipientDidstringRecipient's DID
chatIdstringChat identifier
recipientPubKeystringRecipient's chat public key
var items = new List<AlbumImageInput>
{
    new { ImageData = img1, Width = 800, Height = 600 },
    new { ImageData = img2, Width = 1024, Height = 768 }
};
await sdk.SendAlbumMessageAsync(recipientWallet, items, senderDid, recipientDid, chatId, recipientPubKey);

Message Actions

SendEditMessageAsync Task<bool>

Sends a message edit notification.

await sdk.SendEditMessageAsync(recipientWallet, messageId, newContent, recipientPubKey, recipientDid, chatId);
SendDeleteMessageAsync Task<bool>

Sends a message delete notification.

await sdk.SendDeleteMessageAsync(recipientWallet, messageId, recipientPubKey, recipientDid, deletedBy, chatId);
SendPinMessageAsync Task<bool>

Sends a message pin/unpin notification.

await sdk.SendPinMessageAsync(recipientWallet, targetMessageId, isPinned, chatId, recipientPubKey, recipientDid);
SendReactionMessageAsync Task<bool>

Sends a reaction (emoji) on a message. reactionKey is a Lottie asset key like "like", "love", "haha".

await sdk.SendReactionMessageAsync(
    recipientWallet, targetMessageId, "love", ReactionAction.Add, 
    chatId, recipientPubKey, recipientDid);
IsGroupChatAsync Task<bool>

Check if a chatId corresponds to a group chat.

bool isGroup = await sdk.IsGroupChatAsync(chatId);
