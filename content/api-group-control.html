Group Control Operations

Create groups, manage members, update settings, and handle group metadata.

Setup

SetGroupHandler void

Injects the group message handler for processing incoming group control messages. Call once after DI resolves all singletons.

sdk.SetGroupHandler(groupHandler);
GenerateGroupId string

Generates a unique group ID prefixed with "dc".

string groupId = sdk.GenerateGroupId(); // e.g., "dc1a2b3c..."

Group Creation

SendGroupCreateAsync Task<bool>

Sends a group creation message to all initial members with encrypted sender keys.

ParameterTypeDescription
createDataGroupCreateMessageDataGroup creation payload
senderDidstringCreator's DID
var createData = new GroupCreateMessageData
{
    GroupId = groupId,
    GroupName = "My Group",
    Members = initialMembers,
    Admins = adminList
};
await sdk.SendGroupCreateAsync(createData, senderDid);

ParameterTypeDescription

SendGroupMemberRemoveAsync Task<bool>

Sends a group member remove notification to all remaining members.

ParameterTypeDescription
removeDataGroupMemberRemoveMessageDataMember remove payload
senderDidstringRemover's DID
await sdk.SendGroupMemberRemoveAsync(removeData, senderDid);
SendGroupAdminUpdateAsync Task<bool>

Sends a group admin update message (promote/demote) to all members.

ParameterTypeDescription
groupIdstringGroup identifier
requesterDidstringRequester's DID
targetMemberDidstringTarget member's DID
promoteToAdminboolTrue to promote, false to demote
stateVersionintCurrent state version
memberDidsList<string>All member DIDs for broadcast
await sdk.SendGroupAdminUpdateAsync(
    groupId, requesterDid, targetMemberDid, promoteToAdmin: true, 
    stateVersion, memberDids
);
