Message Types

Complete reference for the MessageContentType enum and its associated MessageDataBase payload types. Every message sent or received carries one of these types in its MessagePayload.Type field.

Content Messages

0
Text
Plain UTF-8 text message with optional formatting.
Data type: TextMessageData Fields: Content, FormattedContent
1
Image
Single image stored on IPFS. Includes thumbnail, dimensions, and optional caption.
Data type: ImageMessageData Fields: IpfsCid, ThumbnailCid, Width, Height, Caption
2
Audio
Voice message or audio clip stored on IPFS.
Data type: AudioMessageData Fields: IpfsCid, DurationSeconds, WaveformData
3
Video
Video with thumbnail stored on IPFS.
Data type: VideoMessageData Fields: IpfsCid, ThumbnailCid, Width, Height, DurationSeconds
4
File
Arbitrary file attachment stored on IPFS.
Data type: FileMessageData Fields: IpfsCid, Filename, MimeType, FileSizeBytes
5
Location
GPS coordinates with optional venue name and address.
Data type: LocationMessageData Fields: Latitude, Longitude, VenueName, Address
7
Sticker
Animated or static sticker from a sticker pack.
Data type: StickerMessageData Fields: StickerId, PackId, IpfsCid
46
Gif
Animated GIF stored on IPFS or linked via URL.
Data type: ImageMessageData (with IsAnimated = true)
60
Album
Multiple images sent as a single grouped message.
Data type: AlbumMessageData Fields: Images[], Caption
47
Poll
Interactive poll with multiple choice options.
Data type: PollMessageData Fields: Question, Options[], ExpiresAt

Social Features

Data type: MessageReactionMessageData Fields: TargetMessageId, Emoji, IsRemoval
38
MessageEdit
Replaces the content of a previously sent message.
Data type: MessageEditMessageData Fields: TargetMessageId, NewContent, EditedAt
39
MessageDelete
Requests deletion of a sent message on all devices.
Data type: MessageDeleteMessageData Fields: TargetMessageId, DeleteForEveryone
40
ThreadReply
A reply that starts or continues a message thread.
Data type: ThreadReplyMessageData Fields: ThreadId, Content
41
ForwardMessage
A message forwarded from another conversation.
Carried in MessagePayload.IsForwarded + ForwardedFromDid
42
MessagePin
Pins or unpins a message in the conversation.
Data type: MessagePinMessageData Fields: TargetMessageId, IsPinned

WebRTC Call Signaling

These types are sent via SendImmediateAsync (fire-and-forget, no ACK queue) to minimise call setup latency.

19
CallOffer
Initiates a WebRTC call. Contains the SDP offer.
Data type: CallOfferMessageData Fields: CallId, SdpOffer, IsVideo
20
CallAnswer
Accepts a call. Contains the SDP answer.
Data type: CallAnswerMessageData Fields: CallId, SdpAnswer
59
CallRinging
Recipient device is ringing sent as early indication before answer.
Data type: CallRingingMessageData Fields: CallId
21
ICECandidate
WebRTC ICE candidate for NAT traversal negotiation.
Data type: ICECandidateMessageData Fields: CallId, Candidate, SdpMid, SdpMLineIndex
22
IPExchange
Direct IP address hint for peer-to-peer connectivity.
Data type: IPExchangeMessageData Fields: CallId, IpAddress, Port
23
CallTermination
Ends an active call. Includes the termination reason.
Data type: CallTerminationMessageData Fields: CallId, Reason
24
GroupCallUpdate
Status update for a group call (member joined, left, or call ended).
Data type: GroupCallUpdateMessageData
25
CallQuality
Real-time call quality metrics report (packet loss, jitter, latency).
Data type: CallQualityMessageData

Contact & Presence

15
ContactRequest
Initiates a contact request. Contains sender's public key and profile stub.
Data type: ContactRequestMessageData Fields: SenderDid, SenderPublicKey, Message
16
ContactResponse
Accepts or rejects a contact request. On accept, returns the responder's chat public key.
Data type: ContactResponseMessageData Fields: Accepted, ChatPublicKey
17
ProfileUpdate
Pushes latest profile data (name, avatar CID, chat public key) to a contact.
Data type: ProfileUpdateMessageData Fields: Username, DisplayName, AvatarIpfsCid, ChatPublicKey, UpdatedFields
18
ProfileRequest
Asks a contact to re-send their current profile.
Data type: ProfileRequestMessageData Fields: RequestedFields, RequestReason
12
ContactPresenceRequest
Pings a contact for their current online/away status.
Data type: ContactPresenceRequestMessageData
13
ContactPresenceResponse
Responds with current presence state (online, away, DND).
Data type: ContactPresenceResponseMessageData Fields: Status, LastSeen
14
TypingIndicator
Notifies the recipient that the sender is currently composing a message.
Data type: TypingIndicatorMessageData Fields: IsTyping

Key Management

10

Challenge/response for safety-number verification (QR code scanning).

Handling Incoming Messages

void HandleMessage(DecryptedMessage msg)
{
    switch (msg.Payload.Type)
    {
        case MessageContentType.Text:
            var text = (TextMessageData)msg.Payload.Data;
            ShowMessage(text.Content);
            break;

        case MessageContentType.Image:
            var img = (ImageMessageData)msg.Payload.Data;
            LoadImage(img.IpfsCid, img.ThumbnailCid);
            break;

        case MessageContentType.KeyRotation:
            var kr = (KeyRotationMessageData)msg.Payload.Data;
            UpdateContactPublicKey(msg.SenderDid, kr.NewChatPublicKey);
            break;

        case MessageContentType.CallOffer:
            var offer = (CallOfferMessageData)msg.Payload.Data;
            ShowIncomingCallUI(offer.CallId, offer.IsVideo);
            break;

        // ... handle other types
    }
}
