DChat SDK API Reference Presence

Presence Operations

Request and respond to online status (presence) for contacts.

Overview

Presence messages are ephemeral - they are not stored on the blockchain or in message history. They provide real-time online status indication between contacts.

Methods

SendPresenceRequestAsync Task<bool>

Sends a presence request to get contact's online status. Ephemeral - not stored.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
recipientPubKeystringRecipient's identity public key
recipientDidstringRecipient's DID
senderDidstringSender's DID
triggerTypestringTrigger reason (e.g., "auto_chat_open", "manual")
await sdk.SendPresenceRequestAsync(
    recipientWallet, recipientPubKey, recipientDid, senderDid, 
    triggerType: "auto_chat_open"
);
SendPresenceResponseAsync Task<bool>

Sends a presence response with current online status.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
recipientPubKeystringRecipient's identity public key
recipientDidstringRecipient's DID
senderDidstringSender's DID
statusPresenceStatusOnline, Offline, Away, Busy
lastActiveFuzzystring?Fuzzy last active time (e.g., "recently", "5m ago")
await sdk.SendPresenceResponseAsync(
    recipientWallet, recipientPubKey, recipientDid, senderDid,
    PresenceStatus.Online, lastActiveFuzzy: "recently"
);

PresenceStatus Enum

ValueDescription
OnlineUser is actively online
OfflineUser is not connected
AwayUser is online but idle
BusyUser is online but busy
