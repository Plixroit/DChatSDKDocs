DChat SDK API Reference Media & IPFS

Media & IPFS Operations

Upload, download, encrypt, and decrypt media files using IPFS distributed storage.

Media Operations

EncryptSignAndUploadMediaAsync Task<string>

Encrypts media with chat key, signs with identity key, and uploads to IPFS. Returns CID.

ParameterTypeDescription
mediaDatabyte[]Media file bytes
senderDidstringSender's DID
senderWalletstringSender's wallet address
recipientPubKeystringRecipient's chat public key
progressIProgress<double>?Optional progress callback (0.0-1.0)
string cid = await sdk.EncryptSignAndUploadMediaAsync(
    mediaData, senderDid, senderWallet, recipientPubKey, uploadProgress
);
DownloadVerifyAndDecryptMediaAsync Task<byte[]>

Downloads media from IPFS, verifies signature, and decrypts with chat key.

ParameterTypeDescription
ipfsCidstringIPFS content identifier
senderDidstringSender's DID
senderIdentityPubKeystringSender's identity public key
senderChatPubKeystringSender's chat public key
receiverDidstringReceiver's DID
mnemonicstringMnemonic for signing request
byte[] media = await sdk.DownloadVerifyAndDecryptMediaAsync(
    ipfsCid, senderDid, senderIdentityPubKey, senderChatPubKey, receiverDid, mnemonic
);
ComputeMediaHash string

Computes SHA-256 hash of media data for integrity verification.

string hash = sdk.ComputeMediaHash(mediaData);

IPFS Operations

UploadToIpfsAsync Task<string>

Uploads a file to IPFS. Returns the CID. Multiple overloads available.

// Basic upload
string cid = await sdk.UploadToIpfsAsync(fileData);

// With progress tracking
string cid = await sdk.UploadToIpfsAsync(fileData, progress);

// With filename and mnemonic
string cid = await sdk.UploadToIpfsAsync(fileData, fileName, mnemonic);

// Full options
string cid = await sdk.UploadToIpfsAsync(fileData, fileName, mnemonic, progress);
DownloadFromIpfsAsync Task<byte[]>

Downloads a file from IPFS.

ParameterTypeDescription
cidstringIPFS content identifier
mnemonicstringMnemonic for signing request
progressIProgress<double>?Optional progress callback
byte[] data = await sdk.DownloadFromIpfsAsync(cid, mnemonic, downloadProgress);
CheckIpfsAvailabilityAsync Task<bool>

Checks if an IPFS CID is available and downloadable.

bool available = await sdk.CheckIpfsAvailabilityAsync(ipfsCid);

Avatar Operations

EncryptAvatarAndUploadToIpfsAsync Task<(string ipfsCid, string avatarKey)>

Encrypts avatar with AES-256-GCM and uploads to IPFS. Returns CID and base64 key. The key must be shared via E2E encrypted messages for contacts to decrypt.

(string cid, string key) = await sdk.EncryptAvatarAndUploadToIpfsAsync(avatarBytes);
// Share 'key' with contacts via encrypted message
DecryptAvatarFromBytesAsync Task<byte[]>

Decrypts an AES-256-GCM encrypted avatar blob using the provided key. Blob format: nonce(12) + ciphertext(N) + tag(16).

byte[] avatar = await sdk.DecryptAvatarFromBytesAsync(encryptedBlob, avatarKeyBase64);
