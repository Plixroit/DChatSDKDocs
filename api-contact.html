Contact & Profile Operations

Manage contacts, send contact requests, and handle profile data.

Contact Operations

SendContactRequestAsync Task<bool>

Sends a contact request to another user after QR code scan or manual entry.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
recipientPubKeystringRecipient's identity public key
recipientDidstringRecipient's DID
senderDidstringSender's DID
displayNamestringSender's display name
chatPubKeystringSender's chat public key
personalMessagestring?Optional personal message
avatarDatabyte[]?Optional avatar bytes
await sdk.SendContactRequestAsync(
    recipientWallet, recipientPubKey, recipientDid, senderDid, displayName, chatPubKey,
    personalMessage: "Hi!", avatarData: null
);
SendContactResponseAsync Task<bool>

Sends a response to a contact request (accept, decline, or block).

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
recipientPubKeystringRecipient's identity public key
recipientDidstringRecipient's DID
senderDidstringSender's DID
statusContactRequestStatusAccepted, Declined, or Blocked
displayNamestringSender's display name
chatPubKeystringSender's chat public key
await sdk.SendContactResponseAsync(
    recipientWallet, recipientPubKey, recipientDid, senderDid,
    ContactRequestStatus.Accepted, displayName, chatPubKey
);

Profile Operations

SendProfileRequest Task<bool>

Sends a profile request message to fetch profile data from a recipient.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
recipientPubKeystringRecipient's identity public key
recipientDidstringRecipient's DID
requestReasonstringReason for request
requestedFieldsList<string>Fields to request (e.g., "avatar", "status")
await sdk.SendProfileRequest(
    recipientWallet, recipientPubKey, recipientDid, 
    requestReason, requestedFields
);
SendProfileUpdateAsync Task<bool>

Sends a profile update message with user profile information.

ParameterTypeDescription
recipientWalletstringRecipient's wallet address
recipientPubKeystringRecipient's identity public key
recipientDidstringRecipient's DID
senderDidstringSender's DID
senderWalletstringSender's wallet address
senderChatPubKeystringSender's chat public key
usernamestringUsername
displayNamestringDisplay name
statusMessagestring?Optional status message
avatarDatabyte[]?Optional avatar bytes
avatarIpfsCidstring?Optional avatar IPFS CID
await sdk.SendProfileUpdateAsync(
    recipientWallet, recipientPubKey, recipientDid, senderDid, senderWallet, senderChatPubKey,
    username, displayName, statusMessage, avatarData, avatarIpfsCid
);
